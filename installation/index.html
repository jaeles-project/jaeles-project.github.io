Installation

Jaeles installation

Make sure Go installed and simply execute the following command

go get -u github.com/jaeles-project/jaeles

Inittialize Signature and Web UI

This gonna pull latest signature and UI build from default repo.

jaeles config -a init

Load new signature to Database

Just save new signature to anywhere then load it by the command below.

jaeles config -a reload -F /tmp/custom-signatures/

with /tmp/custom-signatures/ is your custom signature folder.

Or you can just use new signature directly by using this command

jaeles scan -s new_sign.yaml -u example.com
